# 伴学教室视觉精修第 11 轮：让每个角色拥有自己的工作痕迹

## 目标

继续把角色从“六个相似的插画人物”推进为“在课堂里承担不同职责的智能小组成员”。每个工位通过一个低干扰的职责标记和状态反馈建立识别度，保留课堂环境与人物动画的主导地位。

## 本轮实现

- 为六类职责增加工位上的无文字视觉标记：翻页、灵感灯泡、放大镜、清单、评审屏幕和记录本。
- 让职责标记与真实 runtime 状态联动：准备中和发言时轻微呼吸，等待时降低饱和度，刚完成时出现绿色完成反馈。
- 维持现有角色的发型、肤色、服装、道具、姿态和动作差异；新增标记只作为远距离识别锚点，不增加常驻说明文字。
- 将职责标记加入 reduced-motion 降级，确保辅助功能设置下仍保留静态识别信息。

## 验收

- 角色不被侧栏或文字卡片替代，工位标记服务于人物而不是成为新的面板。
- 任意角色在缩小后的课堂画面中，都能通过人物姿态 + 工位道具 + 标记快速区分职责。
- 标记的动态只表达准备/发言/完成等真实状态，不制造虚假的后台活动。
- 类型检查、目标 ESLint、目标测试和生产构建全部通过。
