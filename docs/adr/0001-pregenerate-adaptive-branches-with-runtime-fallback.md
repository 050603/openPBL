# ADR-0001：自适应分支采用预生成主路径与运行时兜底

## 状态

Accepted

## 背景

完全按需生成导致教师无法在发布前检查内容，学生触发后还需等待，课堂中生成失败会直接
中断学习。完全预生成虽然消除等待，但资源缺失时没有恢复路径。

## 决策

主课堂生成完成后生成所有已确认的自适应分支，并将课堂 ID 保存到分支大纲。学生触发时
直接复用该课堂；资源未准备好或不可用时，复用现有按需生成管线作为兜底。

## 结果

### 正面

- 教师可以预览和控制最终内容。
- 学生正常路径无生成等待。
- 继续复用现有生成、TTS、互动和播放器。
- 单个分支故障不会阻塞主课堂。

### 负面

- 备课生成时间和模型成本增加。
- 分支修改后需要重新生成对应资源。

### 中性

- 运行时仍保留按需生成代码，但只作为容错路径。

## 备选方案

- 完全按需生成：教师不可审、课堂等待明显。
- 只预生成无兜底：运行期可靠性不足。
